Свойства умножения

Свойства умножения: Переместительное свойство умножения: Произведение двух чисел не изменяется при перестановке множителей: a · b = b · а Сочетательное свойство умножения: Чтобы умножить число на произведение двух чисел, можно сначала умножить его на первый множитель, а потом полученное произведение умножить на второй множитель: a · (b · с) = (а · b) · c. Свойство умножения на единицу: Сумма n слагаемых, каждое из которых равно 1, равна 1. Свойство умножения на ноль: Сумма n слагаемых, каждое из которых равно нулю, равна нулю: 0 · n = 0.
